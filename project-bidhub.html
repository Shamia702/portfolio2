BidHub

View live site ↗ GitHub README ↗

BidHub is a student-only auction house built for the Noroff Auction API as my Semester Project 2. Users register with their Noroff student email, create listings, and place bids using credits instead of real money.

The brief

The exam asked for a complete auction platform where users can list items and bid on each other's listings. The goal was to build a full application with authentication, profiles, and a working bidding system on top of a real API, and to handle the different states between logged-out and logged-in users.

How it works

Logged-out visitors can browse the landing page, view all auctions, search listings, and open a single listing to see its details, but they must log in to bid. Once registered with a Noroff student email, users get credits shown in the header, a full profile, and the ability to create, edit and delete their own listings and place bids on others.

Main features

  • Authentication: register and log in with a Noroff student email.
  • Credits: each user has credits, shown in the header on every page.
  • Listings: create, edit and delete listings (owner only).
  • Bidding: place bids, with validation so bids must beat the current highest.
  • Profiles: avatar, banner, bio, credits and stats, plus "My listings" and "My bids" pages.
  • Search & countdown: search listings and see live time remaining on each auction.
