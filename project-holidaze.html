Holidaze

View live site ↗ GitHub README ↗

Holidaze is an accommodation booking application built with React for my exam project. It has two sides: customers can search and book venues, while venue managers can list and manage their own properties, all powered by the Noroff Holidaze API.

The brief

The exam asked for a complete booking platform that serves two different types of users from the same app. The goal was to build a real, responsive application with authentication, a booking system, and a management dashboard, while keeping the customer experience and the manager experience clearly separated.

How it works

Anyone can browse venues, search by name, and open a single venue page to see its details, images and availability calendar. Registered customers can book a venue for selected dates and view or cancel their bookings. Registered venue managers get a dashboard where they can create, edit and delete their own venues and see the bookings made on them.

Main features

  • Venue browsing: search venues with a responsive grid and pagination.
  • Single venue pages: details, images and an availability calendar.
  • Customer bookings: book venues for chosen dates, and view or cancel bookings.
  • Manager dashboard: create, edit and delete venues (owner only).
  • Authentication: register and log in as either a customer or a venue manager.
  • Responsive design: works across mobile, tablet and desktop.
