NextCart

View live site ↗ GitHub README ↗

NextCart is an online shop built with Next.js, React and TypeScript for the JavaScript Frameworks course. It takes a shopper from browsing products all the way through to a checkout success page, using the Noroff Online Shop API for product data.

The brief

The course asked for a working e-commerce application built with a modern JavaScript framework. The goal was to demonstrate component-based architecture, state management for the cart, and a complete user journey rather than a static page.

How it works

Products are fetched from the Noroff Online Shop API and displayed in a responsive grid. Users can search for products, open a product page to see full details, and add items to the cart. The cart keeps track of the items and totals, and the checkout flow ends on a confirmation page. There is also a contact form with validation.

Main features

  • Product listing: products loaded from the API in a responsive grid.
  • Search: filter products by name.
  • Product details: individual product pages with full information.
  • Shopping cart: add items and see the total.
  • Checkout: checkout flow ending on a success page.
  • Contact form: with form validation.
