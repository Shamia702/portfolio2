Talkify

View live site ↗ GitHub README ↗

Talkify is a social media application built with Bootstrap 5 for the CSS Frameworks course. It lets users register, log in, manage their profile, and create, read, update and delete posts powered by the Noroff Social API.

The brief

The course asked for a working social platform built on top of a CSS framework rather than custom styling from scratch. The goal was to show that I can use a component library to build a consistent, responsive interface quickly, while still wiring up real functionality against an API.

How it works

Users authenticate with their Noroff account to register and log in. Once signed in, they can view a feed of posts, open individual posts, follow and unfollow other users, and comment. Post management is full CRUD, users can create new posts, edit their own, and delete them. The interface is built from Bootstrap's grid and components so it stays responsive across screen sizes.

Main features

  • Authentication: register, log in and log out using the Noroff Social API.
  • Profiles: view profiles, with follow and unfollow.
  • Posts: full create, read, update and delete.
  • Comments: comment on posts in the feed.
  • Responsive UI: built with the Bootstrap grid and components.
